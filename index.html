Introduction

MongoDB is a famous NoSQL database used by various organizations to store their data. The data are saved in a JSON-like formed in the hard hard disk in order to be used in the future. As stated by AWS MongoDB provides these features

  • Ad-hoc queries
  • Indexing
  • Replication

Installation

To begin with you have to install the database. The database can be used in various operating systems. Please choose accordingly:

  • Linux
  • MacOS
  • Windows
It is worth noting that the guides are referring to the community edition of the database. If you would like to use MongoDB for your company projects I suggest installing the Enterprise edition.

Getting started

After completing the above steps we can start working with our database. In this guide I am going to show how to perform a wide variety of tasks using the Mongo Shell, although a GUI client might seem easier.

Open shell

To start typing commands a shell has to be opened. To do so you need to open a terminal and type

mongosh

After opening the shell a test database is opened by default. In order to see the database you are working to, you can type

db

Although we could work in test, we are going to create a new one called mydb. The use command will not only create it but it will also change the working enviroment to it

use mydb

Insert data

The data in MongoDB are stored in collections. We could think that the collections are the tables of a relational database, although the data in them are much more flexible. The collections can store different kinds of data, which may not have common fields. So let' s assume we would like to create a new collection called students, in which the name and age of two persons will be stored. By using the insertMany command as shown below a new collection in the chosen database is going to be created with the desired name and it will contain the data of two students

db.students.insertMany([
{name: "Mark",
date-of-birth: 2000
},
{name: "Helen",
date-of-birth: 1999
}
])

A useful feature of MongoDB is that you can assign a list of values in an attribute of an object. To do so you have to give the values like this

attribute: [value1, value2, ...]

Find data

A key functionality of a database is searching for data. To find desired data in a collection when using Mongo, the find command comes in mind, which receives arguments like

{attribute1: value1, attribute2: value2, ...}

So imagine we would like to find all students who were born in 1999.

db.students.find({date-of-birth: 18});

Another important aspect of find is that it can show us all the documents in a collection. To print them on screen type

db.students.find({});

Delete data

Finally we will discuss how to delete a document from a collection using the deleteMany({attribute: value, ...}) command. As an example we will assume that we would like to delete all documents from the collection students, in which the name is Helen. To accomplish this type

db.students.deleteMany({name: "Helen"});
